# VR Sistema Solar

Proyecto en A-Frame que muestra una experiencia interactiva sobre el sistema solar. La aplicación contiene 3 escenas distintas, combinando visualización, interacción y un minijuego


## Escenas

### 1. Sistema Solar
Escena principal donde se representan los planetas del sistema solar en un entorno VR. Permite observarlos en el espacio y explorar el sistema solar

### 2. Vista detallada de Marte
En esta escena, al hacer clic sobre Marte, el planeta se muestra en detalle

### 3. Minijuego: Adivina el planeta
Minijuego en el que el usuario debe adivinar el planeta


## Tecnologías utilizadas

- A-Frame
- JavaScript
